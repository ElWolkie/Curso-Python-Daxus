📁 Proyectos Incluidos


1 Generador Automatizado de Presupuestos (PDF) 📄
Descripción: Herramienta para la creación dinámica de presupuestos profesionales. El sistema recibe entradas del usuario (descripción, horas estimadas, valor de la hora) y genera un PDF formateado con cálculos automáticos.

      Tecnologías: fpdf.
      
      Impacto: Digitalización de procesos administrativos, eliminando el error humano en cálculos manuales.



2 Automatización RPA: Envío de Informes vía Gmail 🤖
Descripción: Implementación de RPA (Robotic Process Automation) para la gestión de comunicaciones. El script automatiza el uso del hardware (teclado y ratón) para navegar, redactar y enviar informes a través de Gmail de manera autónoma.

      Conceptos clave: Simulación de periféricos, control de flujos de trabajo y automatización de interfaz de usuario.
      
      Nota: Ideal para procesos donde no existen APIs directas y se requiere interactuar con la UI.



3 Análisis de Datos e Inteligencia de Negocios (BI) 📊
Descripción: Procesamiento de grandes volúmenes de datos provenientes de archivos Excel para identificar tendencias de ventas y facturación.

    Funcionalidades:
    
      Análisis exploratorio con Pandas.
      
      Agrupamiento de datos por tiendas y productos.
      
      Generación de dashboards interactivos con Plotly.
  
Visualización: Incluye histogramas dinámicos que permiten segmentar la facturación por métodos de pago y ubicación.

4 IA: Seguimiento de Manos en Tiempo Real (Visión Artificial) 🖐️
Descripción: Desarrollo de una Inteligencia Artificial capaz de detectar y rastrear extremidades humanas a través de la webcam.

      Tecnologías: OpenCV, cvzone y MediaPipe.

Funcionamiento: El sistema procesa video en tiempo real, identificando puntos clave de la mano y dibujando una malla de seguimiento (landmarks) con alta precisión.
